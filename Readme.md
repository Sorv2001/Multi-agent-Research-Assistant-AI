# 🌿 AI Research Assistant

A **multi-agent AI system** that automates the entire research workflow:
🔍 Search → 📄 Scrape → 📝 Report → 📊 Critic Feedback

---

## 🚀 Overview

This project uses **LangChain + Mistral AI** to build a smart research assistant that can:
- Find relevant information from the web  
- Extract meaningful content  
- Generate structured reports  
- Evaluate the quality of the report  

It is designed for **corporate use cases** like market research, decision-making, and documentation.

---

## 🧠 Architecture

The system follows a **multi-agent pipeline**:

1. **Search Agent**  
   Retrieves relevant information based on the topic  

2. **Reader Agent**  
   Extracts detailed content from selected sources  

3. **Writer Chain**  
   Generates a structured report with:
   - Introduction  
   - Key Findings  
   - Conclusion  
   - Sources  

4. **Critic Chain**  
   Evaluates the report and provides:
   - Score  
   - Strengths  
   - Areas for improvement
